## Die 9 kreativen Raumtypen

### 1. Project Spaces — Geteilte Schöpfung
**Was:** Ein temporärer, fokussierter Raum für eine konkrete Zusammenarbeit.
**Wie lange:** So lange das Projekt dauert — dann wird der Raum still.
**Wer:** 2–6 Creators, bewusst eingeladen.
**Nicht:** Ein Projekt-Management-Tool. Keine Tickets, keine Deadlines-Tyrannei.
**Atmosphäre:** Ruhig, klar, fokussiert. Wie ein gemeinsames Atelier.
```
Elemente: shared mood · timeline · collaborative notes
          reference board · resonance log · contribution visibility
```

### 2. Resonance Rooms — Atmosphärische Begegnung
**Was:** Temporäre thematische Räume, in denen Creators still miteinander resonieren.
**Wie lange:** 24h bis 2 Wochen — dann verschwinden sie ruhig.
**Wer:** Offen (bis max 20), aber thematisch gefiltert.
**Nicht:** Ein Gruppen-Chat. Keine Nachrichten-Flut.
**Atmosphäre:** Langsam, stimmungsvoll, bewusst begrenzt.
```
Beispiele:
  „nächtliche elektronische Klangwelten" (48h)
  „ruhige visuelle Experimente dieser Woche"
  „lokale Naturmaterialien als Werkstoff"
```

### 3. Local Creative Circles — Regionale Verbindung
**Was:** Stille, lokale kreative Netzwerke in einer Stadt oder Region.
**Wie lange:** Dauerhaft, aber ruhig (keine tägliche Aktivität nötig).
**Wer:** Creators aus derselben Stadt/Region.
**Nicht:** Stadtgruppe auf Facebook.
**Atmosphäre:** Vertraut, lokal verankert, keine globale Konkurrenz.
```
Elemente: nearby resonance · city clusters · local bridge creators
          regional discovery boost · lokale Begegnungsräume
```

### 4. Mentorship Circles — Kreative Weitergabe
**Was:** Ein stiller Raum der Weitergabe zwischen erfahrenen und neuen Creators.
**Wie lange:** 3–6 Monate, bewusst befristet.
**Wer:** 1 erfahrener Creator + 1–3 Newcomer.
**Nicht:** Coaching-Marktplatz. Kein Rating-System für Mentoren.
**Atmosphäre:** Großzügig, ruhig, ohne Leistungsdruck.
```
Elemente: slow check-ins · resonance sharing · trust continuity
```

### 5. Thematic Creative Groups — Gemeinsame Ausrichtung
**Was:** Langfristige Gemeinschaften rund um eine kreative Haltung oder Praxis.
**Wie lange:** Dauerhaft, aber wandelbar.
**Wer:** Offen, aber thematisch klar.
**Nicht:** Community für alle. Klar fokussiert.
**Atmosphäre:** Identitätsstiftend, nicht ausschließend.
```
Beispiele:
  „handgemacht und regional"
  „digitale Ruhe" (slow tech creators)
  „interdisziplinäres Schaffen"
```

### 6. Slow Creative Sessions — Gemeinsame Vertiefung
**Was:** Zeitlich begrenzte, bewusst langsame Kreativ-Sessions.
**Wie lange:** 2–6 Stunden, dann vorbei.
**Wer:** 2–8 Creators.
**Nicht:** Sprint oder Hackathon. Das Gegenteil.
**Atmosphäre:** Meditativ, tief, ohne Produktionsdruck.
```
Elemente: shared ambient state · slow contribution rhythm
          presence without performance
```

### 7. Creative Journeys — Langfristige Entwicklung
**Was:** Der persönliche kreative Entwicklungsweg über Zeit.
**Wie lange:** Dauerhaft — die Geschichte eines Creators.
**Wer:** Jeder Creator hat seine eigene Journey.
**Nicht:** Level-System oder XP. Nie gamifiziert.
**Atmosphäre:** Reflektiv, ehrlich, nicht wertend.
```
Elemente: collaboration history · resonance timeline
          interdisciplinary transitions · community contributions
```

### 8. Interdisciplinary Bridges — Über Grenzen hinaus
**Was:** Räume wo Creators aus völlig verschiedenen Feldern sich begegnen.
**Wie lange:** Temporär, als Einladung.
**Wer:** Bridge-Creators die mehrere kreative Welten verbinden.
**Nicht:** Zufällige Mischung. Bewusst kuratiert.
**Atmosphäre:** Neugierig, offen, überraschend.
```
Bridge-Typen: visual × sonic · local × digital · craft × technology
```

### 9. Resonance Memory — Kreative Geschichte
**Was:** Das kollektive Gedächtnis von Zusammenarbeit und Resonanz.
**Wie lange:** Dauerhaft gespeichert, ruhig sichtbar.
**Wer:** Alle Beteiligten einer Zusammenarbeit.
**Nicht:** Feed-Algorithmus-Futter. Privates Gedächtnis.
**Atmosphäre:** Warm, ehrend, nicht ausgestellt.
```
Elemente: shared project history · resonance moments
          trust continuity · collaboration artefacts
```

## Gemeinsame Prinzipien aller Creative Spaces

| Prinzip | Was das bedeutet |
|---------|-----------------|
| **Ruhig** | Kein Notification-Druck. Keine Pflicht zur täglichen Aktivität. |
| **Temporär wenn nötig** | Räume dürfen verschwinden — das ist gut. |
| **Atmosphärisch** | Jeder Raum hat eine eigene Stimmung. |
| **Nicht-gamifiziert** | Kein Ranking, kein Score, kein Level. |
| **Vertrauensbasiert** | Betritt man nur mit Einladung oder gemeinsamen Werten. |
| **Lokal verankert** | Reale Ort-Verbindungen werden gestärkt. |
| **Tief statt breit** | 3 bedeutungsvolle Verbindungen > 300 flüchtige. |
