# HUI — COLLABORATIVE PROCESS MAP
**Phase 7B.1 — Wie kreative Zusammenarbeit wirklich verläuft**

---

> „Gemeinsame Kreativität ist kein Sprint.
> Sie ist ein Gespräch — das manchmal Pausen braucht."

---

## Die 7 Wahrheiten kreativer Zusammenarbeit

### 1. Sie ist nicht linear
Kein Projekt folgt dem Weg: Idee → Planung → Umsetzung → Fertig.
Kreative Zusammenarbeit spiralt. Sie kehrt zurück.
Eine verworfene Idee wird Wochen später die richtige sein.

### 2. Pausen sind Teil des Prozesses
Stille zwischen zwei Nachrichten ist keine Inaktivität.
Sie ist Nachdenken. Verarbeiten. Lassen.
Ein System das Pausen bestraft zerstört kreative Tiefe.

### 3. Asynchronität ist natürlich
Nicht jeder kreative Moment geschieht gleichzeitig.
Einer denkt nachts. Der andere morgens.
Co-Kreation bedeutet nicht: gleichzeitig online sein.

### 4. Iteration ist keine Schwäche
Fünf Versuche bis etwas stimmt sind nicht Ineffizienz.
Sie sind der kreative Prozess selbst.
Keine "Version 1.0"-Erwartungen.

### 5. Emotionale Dynamik ist real
Begeisterung. Zweifel. Erschöpfung. Erneuerung.
Kreative Zusammenarbeit hat eine emotionale Temperatur.
Sie schwankt — und das ist normal.

### 6. Kreative Spannungen sind produktiv
Verschiedene Sichtweisen erzeugen Reibung.
Reibung erzeugt Wärme.
Wärme erzeugt neues Denken.
Ein System das Spannung eliminiert eliminiert Kreativität.

### 7. Übergänge brauchen Raum
Der Moment wo ein Projekt in eine neue Phase tritt
braucht Bewusstsein — nicht automatischen Status-Wechsel.
*"Wir sind nicht mehr am Anfang"* ist ein bedeutsamer Moment.

---

## Die 8 Co-Creation Zustände

```
exploring      → Wir suchen noch. Alles ist offen.
deepening      → Eine Richtung zeigt sich. Wir gehen tiefer.
experimenting  → Wir probieren aus. Scheitern ist erlaubt.
refining       → Etwas Konkretes entsteht. Es wird klarer.
resting        → Bewusste Pause. Alle Beteiligten atmen durch.
transitioning  → Das Projekt wechselt seinen Charakter.
finishing      → Das Werk sucht seinen Abschluss.
dormant        → Das Projekt ruht — und könnte wieder erwachen.
```

**Wichtig:** Kein linearer Fortschritt.
`exploring → deepening → resting → experimenting` ist genauso gültig
wie jede andere Reihenfolge.

## Kollaborations-Typen nach Tiefe

| Typ | Dauer | Tiefe | Charakteristik |
|-----|-------|-------|----------------|
| Flash | Stunden | leicht | Spontaner gemeinsamer Impuls |
| Session | Tage | mittel | Gemeinsame kreative Session |
| Project | Wochen | tief | Konkretes gemeinsames Werk |
| Journey | Monate | sehr tief | Langfristige kreative Partnerschaft |
| Ongoing | Offen | permanent | Wiederkehrende kreative Verbindung |
